# Portfolio App

Sitio de portafolio personal construido con Next.js, React y TypeScript.

## Tecnologias

- Next.js 16
- React 19
- TypeScript
- Framer Motion

## Requisitos

- Node.js 20+
- pnpm

## Como correr el proyecto

1. Instala dependencias:

```bash
pnpm install
```

2. Inicia el servidor de desarrollo:

```bash
pnpm dev
```

3. Abre en el navegador:

```text
http://localhost:3000
```

## Comandos utiles

- `pnpm dev`: corre en modo desarrollo.
- `pnpm build`: genera build de produccion.
- `pnpm start`: levanta la app en modo produccion.
- `pnpm lint`: ejecuta ESLint.
